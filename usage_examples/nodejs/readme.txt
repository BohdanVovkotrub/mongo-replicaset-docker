NodeJS is required !!!
Mongodb Replica Set must already be running !!!


To run the example, follow these steps:


1. Go to this folder:

    cd <path-to-this-folder>

2. Install dependencies:

    npm install 

3. Once the dependencies have been installed, create and configure ".env" file.
    You can copy the contents of ".env.example" file to ".env" file and change the values according to your needs.

4. Run this example:

    node index.js
